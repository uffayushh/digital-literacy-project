## what is this project even

so basically i got selected as a Student Digital Ambassador which sounds fancy but basically means i have to help other students not mess up their online presence lol. the whole project is about creating a portfolio that shows i understand digital literacy - like how to use tech safely, communicate properly online, avoid getting scammed, that kind of stuff.

its 5 different tasks with 20 marks each so 100 total. im not gonna lie i thought it would be boring but i actually learned some pretty useful things. like i didnt realize how easy it is to fall for phishing emails or how bad it is to reuse passwords everywhere.

## task 1 - made an infographic

created a digital literacy infographic on canva. it has stuff about:
- what digital literacy means (not just knowing how to use a phone lol)
- tools we actually use as students (gmail, zoom, google docs, notion)
- how to stay safe online (dont click random links, strong passwords etc)

canva was super easy to use honestly. just drag and drop stuff and it looks decent. took like 45 mins to make something that looked decent.

## task 2 - professional profiles setup

set up accounts on:
- GitHub (made a profile readme with my name and stuff)
- LinkedIn (filled in my education section)
- Kaggle (cuz why not)

basically im building my online presence for when i apply for internships and jobs. kind of important apparently. i realized my instagram is absolutely not appropriate for anyone professional to see lmao so im keeping those profiles separate.

## task 3 - coding and collaboration platforms

did some beginner coding challenges on hackerrank. honestly it was kinda fun? like solving problems even if theyre simple. 

also made a google form with 5 questions about digital literacy that could actually be used to quiz my batch. it has multiple choice and short answer questions. the responses automatically collect in a sheet which is pretty cool.

---

## task 4 - emails and social media checklist

wrote 2 professional emails:
1. asking a professor for an assignment extension (something we all need lol)
2. expressing interest in an internship

also created a do's and dont's checklist for social media. like dont post ur aadhar number on instagram for clout type thing.

---

## task 5 - cybercrime awareness

chose phishing as my cybercrime topic cuz its actually super relevant to college students. wrote a case study about how it works and whos usually targeted.

also made a prevention checklist with tips on:
- how to not get scammed on upi (important for us indians)
- password security
- public wifi dangers
- how to report stuff to cybercrime portal

included the national cyber crime portal (cybercrime.gov.in) and the helpline 1930 cuz thats actually important to know.
